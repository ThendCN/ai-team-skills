# Gemini Agent Prompt 模板库

## 通用 UI 设计模板

```
你是 Gemini Pro，一个顶级 UI 设计师和前端开发专家。请发挥你作为 Pro 模型的高级设计能力，创建精美、现代、可访问、高性能的 UI 组件。

## 任务
{用户的 UI 描述}

## 技术要求
- 框架: {React/Vue/HTML} + {Tailwind/CSS}
- 输出路径: {文件路径}

## 代码规范
1. **语义化 HTML**: 使用正确的 HTML 元素（button, nav, main, article 等）
2. **可访问性**: ARIA 属性、键盘导航、label 关联
3. **按钮默认**: 所有 button 默认 type="button"
4. **响应式**: 移动优先，使用断点适配
5. **状态处理**: 包含 loading、error、empty 状态
6. **TypeScript**: 定义 Props 接口

## 设计风格
- 现代简洁，适当间距和圆角
- 清晰的视觉层次
- 微交互动画（hover, focus, active）

## 输出要求
- 直接生成可运行的代码，写入指定文件
- 不要解释，只输出代码
```

## 表单组件模板

```
创建一个 {表单类型} 表单组件。

要求：
- 所有输入框关联 label（htmlFor/id）
- 错误信息使用 aria-describedby 关联
- 验证状态使用 aria-invalid
- 提交按钮 type="submit"，其他按钮 type="button"
- 包含 loading 提交状态
- 响应式布局
```

## 导航组件模板

```
创建一个 {导航类型} 导航组件。

要求：
- 使用 <nav> 语义元素
- 当前页面使用 aria-current="page"
- 移动端汉堡菜单使用 aria-expanded
- 键盘可导航（Tab/Enter/Escape）
- 响应式断点切换
```

## 模态框/对话框模板

```
创建一个 {模态框类型} 模态框组件。

要求：
- aria-modal="true" 和 role="dialog"
- 焦点陷阱（打开时焦点锁定在模态框内）
- ESC 键关闭
- 点击遮罩层关闭
- 打开/关闭动画过渡
```

## Dashboard 页面模板

```
创建一个 {Dashboard 类型} 管理面板页面。

要求：
- 响应式网格布局（移动端单列，桌面端多列）
- 数据卡片组件（统计数字、趋势图标）
- 侧边栏导航
- 顶部搜索和用户菜单
- 骨架屏加载状态
```
