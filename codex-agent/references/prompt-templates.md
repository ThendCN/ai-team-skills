# Codex Agent Prompt 模板库

## 通用代码实现模板

```
你是一个高级软件工程师。请完成以下编码任务。

## 任务
{任务描述}

## 项目上下文
- 语言/框架: {技术栈}
- 项目结构: {关键目录和文件}
- 相关文件: {需要参考或修改的文件路径}

## 要求
1. 遵循项目现有的编码风格和约定
2. 添加必要的错误处理
3. 确保类型安全（如适用）
4. 代码应可直接运行

## 输出
将代码写入以下文件：
{文件路径列表}
```

## 流水线模式 - 实现阶段 Prompt 模板

```
你是一个高级全栈工程师。基于已有的 UI 设计代码，实现完整的业务逻辑。

## UI 设计上下文
以下文件包含已生成的 UI 代码（由 Gemini 设计）：
{UI 文件路径列表}

关键设计决策：
{从 UI 代码中提取的关键信息}

## 实现任务
{需要实现的业务逻辑描述}

## 要求
1. 保持 UI 代码的组件结构不变
2. 实现数据获取、状态管理、事件处理
3. 连接 API 端点
4. 添加错误处理和加载状态
5. 确保类型安全

## 输出
修改或创建以下文件：
{文件路径列表}
```
